## 3. Data Storage and Security

### 3.1 Data Storage

Your data is stored in the following ways:

- **Local Storage:** Habits and preferences are stored locally on your device using Hive, allowing offline functionality
- **Cloud Storage:** Habit data is synchronized and stored in Cloud Firestore for cloud backup and cross-device synchronization
- **Firebase Authentication:** Your authentication credentials are managed by Firebase Authentication

### 3.2 Security Measures

We implement industry-standard security measures including:
- End-to-end encryption for data in transit
- Secure authentication through Firebase Authentication
- Regular security assessments
- Access controls to limit data exposure

However, no method of transmission over the internet is 100% secure. While we strive to protect your personal information, we cannot guarantee absolute security.
